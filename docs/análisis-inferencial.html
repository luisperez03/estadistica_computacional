5 Análisis Inferencial

En esta sección nos dedicaremos a mostrar por medio de pruebas estadísticas las hipótesis relacionadas a la incidencia de distintos factores de privaciones sobre el IPM, su influencia y la sensibilidad de ser catalogado como pobre multidimensional a partir de la jefatura del hogar o la ubicación geográfica.

5.1 Incidencia de género en hogares monoparentales sobre el IPM

Dentro de las variables que no puntúan el IPM, se encuentra el género del jefe del hogar y se puede relacionar con si este es un hogar monoparental. Dadas las condiciones, se probará si existe una diferencia entre el caso de que el hogar sea liderado por un hombre o una mujer, ambos solteros.

# Variable jefe copmo factor
df$P6020_jefe <- factor(df$P6020_jefe)

#Hogares monoparentales
df_monoparental <- df %>% 
  filter(is.na(x = P6020_conyuge))

# Crear tabla de contingencia
tabla_pobreza <- table(df_monoparental$P6020_jefe, df_monoparental$POBRE)

# Asignar nombres a filas y columnas
dimnames(tabla_pobreza) <- list(
  Genero = c("Hombre", "Mujer"),   # según codificación de P6020_jefe
  Pobreza = c("No pobre", "Pobre") # según codificación de POBRE
)


# Ver proporciones (porcentajes por fila)

5.2 Regiones y su incidencia en la pobreza

La desigualdad en Colombia también responde a una influencia geográfica, por múltiples factores, algunas regiones son más vulnerables/vulneradas que otras.

Para esta data comparamos el IPM entre regiones usando la prueba no paramétrica de Kruskal Wallis, dado que es una variable continua que avanza de pequeños saltitos y no tiene una forma normal.

df$REGION <- factor(df$REGION)

# Test de Kruskal-Wallis
# Compara el puntaje IPM a través de las diferentes regiones
kruskal_res <- kruskal.test(IPM ~ REGION, data = df)


print(kruskal_res)
## 
##  Kruskal-Wallis rank sum test
## 
## data:  IPM by REGION
## Kruskal-Wallis chi-squared = 2388.8, df = 5, p-value < 2.2e-16

Tras obtener un p-valor inferior a 005 en la prueba Kruska-lWallis podemos decir que existe evidencia estadstica suficiente para afirmar que el IPM presenta variaciones significativas entre las distintas regiones Ahora realizaremos comparaciones mltiples entre regiones para identificar cuales presentan niveles de pobreza significativamente ms altos que otras Utilizando la prueba posthoc DUNN test de la librera FSA que nos permitir establecer un ranking de pobreza regional Adicionalmente usaremos el ajuste de Benjamini-Hochberg para controlar la tasa de falsos descubrimientos y garantizar la robustez de las comparaciones

df$region_nombre <- factor(df$REGION,levels = c("1","2","3","4","5","6"),labels = c("Caribe", "Oriental", "Central", "Pacífica", "Antioquia", "Bogotá"))

if(kruskal_res$p.value < 0.05){
  post_hoc <- dunnTest(IPM ~ region_nombre,
                       data = df,
                       method = "bh") # Ajuste de Benjamini-Hochberg
  print(post_hoc)
}

5.3 Pobreza en cabeceras principales vs centros poblados y rurales

Aquí hablaremos de cmo influye estar cerca o lejos de las ciudades principales en en la intensidad de la pobreza multidimensional El objetivo es evidenciar estadsticamente las razones del desplazamiento voluntario de la ruralidad a las ciudades en busca de oportunidades y mejor calidad de vida

df$P3 <- factor(df$P3)

# Test de Kruskal-Wallis
# Compara el puntaje IPM a través de las diferentes ubicaciones
kruskal_res_P3 <- kruskal.test(IPM ~ P3, data = df)

# resultado
print(kruskal_res_P3)
## 
##  Kruskal-Wallis rank sum test
## 
## data:  IPM by P3
## Kruskal-Wallis chi-squared = 9520.5, df = 2, p-value < 2.2e-16

La prueba arroja un valor-P inferior a 0.05, lo que nos dice que es estadisticamente significativa la diferencia de la intensidad promedio de la pobreza segun la ubicacion del hogar , ahora veamos donde es mas intensa dicha diferencia

df$P3_nombre <- factor(df$P3,levels = c("1","2","3"),labels = c("Cabecera", "Centro poblado", "Rural disperso"))

if(kruskal_res_P3$p.value < 0.05){
  library(FSA)
  post_hoc_P3 <- dunnTest(IPM ~ P3_nombre,
                       data = df,
                       method = "bh") # Ajuste de Benjamini-Hochberg
  print(post_hoc_P3)
}
##                        Comparison         Z       P.unadj         P.adj

Esto muestra que la mayor diferencia es entre las cabeceras y los centros de rurales dispersos, seguido de las cabeceras y los centros rurales, dando evidencia de que la intensidad de la pobreza va aumentando entre ubicaciones del hogar, explicando una parte del desplazamiento voluntario de las personas de estos lugares hacia la ciudad.

5.4 Número de habitantes del hogar vs pobreza multidimensional

Los hogares pueden estar conformados tanto por familias pequeñas, como otras muy grandes ¿Esta concentración de personas afecta la probabilidad de ser calificado como pobre multidimensional? Para ello se generan grupos de hogares según el número de personas para poder así observar la proporción de los calificados como pobres y no pobres.

# categorías de tamaño de hogar
df <- df %>%
  mutate(grupo_hogar = case_when(
    PERSONAS <= 2 ~ "1-2 pers",
    PERSONAS <= 4 ~ "3-4 pers",
    PERSONAS <= 6 ~ "5-6 pers",
    TRUE            ~ "7+ pers"
  ))

df$grupo_hogar <- factor(df$grupo_hogar, 
                                    levels = c("1-2 pers", "3-4 pers", "5-6 pers", "7+ pers"))

# tabla de contingencia
tabla_contingencia <- table(df$grupo_hogar, df$POBRE)
# Asignar nombres más claros a las columnas
colnames(tabla_contingencia) <- c("No pobre", "Pobre")

# Ver proporciones por fila
knitr::kable(round(prop.table(tabla_contingencia, 1),2))

Se observa un aumento en la proporción de ser pobre de hogares según la cantidad de personas que lo conforman, sin embargo apliquemos la prueba chi-cuadrado para definir validar dicha diferencia

# Prueba Chi-cuadrado
chi_resultado <- chisq.test(tabla_contingencia)
